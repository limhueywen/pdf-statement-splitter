User Guide
PDF Statement Splitter — by Lim Huey Wen
No installation required. This tool runs entirely in your browser. Files are processed locally — no data is uploaded to any server.
1
Upload your combined PDF
Drag the file onto the upload area or click to browse. This should be your bulk statement — a single PDF containing all customer accounts.
2
Scan & Detect Accounts
The tool reads every page and tries known label patterns such as Account No :. No typing needed — detection is fully automatic.
3
Review the account list
A table shows every code found and its page count. If something looks wrong, click Edit pattern to set the label manually, then Re-scan.
4
Split the PDF
Click Split into Separate PDFs. One output file is created per account code. Multi-page accounts are grouped correctly.
5
Download your files
Click any card to download that file (e.g. 102001.pdf). Use Download ZIP to receive all files in one archive.
Pattern not detected? Open your PDF, find the label before the account code (e.g. Account No :), and enter it in the Edit Pattern field.
